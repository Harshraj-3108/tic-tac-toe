# 🎮 Tic Tac Toe AI

An interactive Tic Tac Toe game where you can play against the computer 🤖. Built using HTML, CSS, and JavaScript with a clean UI and smooth gameplay.

---
## 🚀 Features

* 🔹 Player vs Computer gameplay
* 🔹 Color-coded moves (🔵 X / 🔴 O)
* 🔹 Winning line highlight
* 🔹 Automatic win/draw detection
* 🔹 Simple and responsive design
---
## 🧠 Tech Stack

* HTML
* CSS
* JavaScript
---
## 🎯 How to Play
1. Click on any empty box to place **X**
2. Computer will automatically place **O**
3. First player to align 3 marks (row/column/diagonal) wins
4. If all boxes are filled and no winner → Draw
